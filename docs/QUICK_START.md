# Quick Start - Naukri AI Bot (Ultra-Optimized)

## 1️⃣ Install & Setup (First Time Only)

```bash
# Clone/navigate to project
cd /Users/koushik/Documents/naukri_ai_bot_automatic_job_apply

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install chromium
```

## 2️⃣ Configure (One Time)

Edit `src/conf.py`:
```python
EMAIL = "your_naukri_email@gmail.com"
PASSWORD = "your_password"
API_KEY = "your_gemini_key"  # Fallback only
MY_EXPERIENCE = 2.5
OLLAMA_MODEL = "qwen2-vl:4b"
USE_OLLAMA_FIRST = True
```

Add resume: `data/Resume.pdf`

## 3️⃣ Start Ollama (Terminal 1)

```bash
ollama serve
```

Wait for:
```
✅ Listening on 127.0.0.1:11434
```

## 4️⃣ Pull Model (Terminal 2, First Time Only)

```bash
ollama pull qwen2-vl:4b
```

Wait for download (~3GB).

## 5️⃣ Run Bot (Terminal 2)

```bash
cd /Users/koushik/Documents/naukri_ai_bot_automatic_job_apply
source venv/bin/activate
python3 src/bot.py
```

## 🛠️ Troubleshooting

### Ollama not responding
```bash
# Check if running
curl http://localhost:11434/api/tags

# Restart
ollama serve
```

### Model not found
```bash
ollama pull qwen2-vl:4b
```

### Bot not filling answers
Check `logs/bot.log`:
```bash
tail -100 logs/bot.log | grep "Error"
```

### Wrong answers still appearing
1. Clear cache: `rm data/qa_store.json`
2. Restart bot: `python3 src/bot.py`

## 📊 Files to Monitor

- **logs/bot.log** — Detailed execution log
- **data/qa_store.json** — Cached Q&A pairs (grows over time)
- **output/external_jobs.json** — Jobs redirected to company sites
